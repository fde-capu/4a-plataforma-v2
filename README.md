### Ex 1. Contagem de aparições de vetor em uma matriz.

O arquivo `bitmap.json` deve conter a matriz de referência.

:: Use: 

- `python3 main.py [1, 2, 3]`
- `python3 main.py 1, 2, 3`
- `python3 main.py 1 2 3`
- `python3 main.py "[1, 2, 3]"`
- `python3 main.py "1, 2, 3"`
- `python3 main.py "1 2 3"`
- `python3 unit_test.py` para rodar testes.

	(Aspas duplas `""` e simples `''` são aceitas.)
