O arquivo `bitmap.json` deve conter a matriz de referência.

Ex 1. Contagem de aparições de vetor em uma matriz (definida pelo arquivo).
	Use: 
	- `python3 bmp_matrix.py [1, 2, 3]`
	- `python3 bmp_matrix.py 1, 2, 3`
	- `python3 bmp_matrix.py 1 2 3`
	- `python3 bmp_matrix.py "[1, 2, 3]"`
	- `python3 bmp_matrix.py "1, 2, 3"`
	- `python3 bmp_matrix.py "1 2 3"`
		(double `""` and single `''` quotes accepted).
